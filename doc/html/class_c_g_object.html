Opis szczegółowy

Definicja klasy CGObject.

Klasa odpowiada za wyświetlanie graficznych odpowiedników obiektów klasy logicznej CObject.

  • Zobacz również
    CObject Klasa jest matką pozostałych klas graficznych: CGDisease, CGDoctor, CGMedicament, CGPatient, .
